# BarberPro

Sistema profissional para agendamento de barbeiros. PWA com React, Vite, Tailwind e Supabase.

## Rodar localmente

```sh
npm i
npm run dev
```

Acesse: http://localhost:8080

## Build e deploy

```sh
npm run build
npm run deploy
```

Deploy envia os arquivos para o servidor via SSH (configurado em `deploy-ssh.ps1`).

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Supabase (auth + banco)
- PWA (Vite PWA)
