SOLUÇÃO IMEDIATA - PAINEL HOSTINGER:

OPÇÃO 1 - CONSOLE WEB:
1. Acesse hpanel.hostinger.com
2. Vá para VPS > Seu VPS (72.60.159.183)
3. Procure por "Console" ou "Terminal Web"
4. Clique para abrir o console no navegador
5. Execute os comandos diretamente no console:
   cd /var/www/barberpro
   pm2 restart barberpro
   pm2 save

OPÇÃO 2 - RESETAR SENHA:
1. No painel Hostinger > VPS > Gerenciar
2. Procure "Resetar Senha Root" ou "Alterar Senha"
3. Confirme o reset
4. Aguarde email com nova senha
5. Use a nova senha para SSH

OPÇÃO 3 - UPLOAD VIA PAINEL:
1. No painel VPS procure por "Gerenciador de Arquivos"
2. Navegue até /var/www/barberpro/
3. Faça upload dos arquivos da pasta 'dist'
4. Reinicie o serviço pelo console

ARQUIVOS PARA UPLOAD:
- Toda a pasta 'dist' (já buildada)
- Substitua os arquivos existentes

SE CONSEGUIR ACESSO POR QUALQUER MÉTODO:
pm2 restart barberpro
pm2 status
pm2 logs barberpro
