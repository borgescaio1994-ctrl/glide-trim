SOLUÇÕES PARA PROBLEMA DE ACESSO SSH:

OPÇÃO 1 - VERIFICAR SENHA NO PAINEL HOSTINGER:
1. Acessar painel da Hostinger
2. Ir para VPS -> Gerenciar
3. Procurar "Senha Root" ou "Resetar Senha"
4. Usar a nova senha para SSH

OPÇÃO 2 - USAR CHAVE SSH:
1. Gerar chave: ssh-keygen -t rsa -b 4096
2. Copiar chave pública: cat ~/.ssh/id_rsa.pub
3. Adicionar no servidor: echo "SUA_CHAVE_PUBLICA" >> ~/.ssh/authorized_keys

OPÇÃO 3 - ACESSO VIA PAINEL HOSTINGER:
1. Usar console web do painel Hostinger
2. Navegar até /var/www/barberpro
3. Fazer upload dos arquivos da pasta 'dist'

OPÇÃO 4 - FTP/SFTP:
1. Configurar cliente FTP (FileZilla)
2. Usar credenciais: root@72.60.159.183
3. Porta: 22 (SFTP) ou 21 (FTP)
4. Fazer upload da pasta 'dist'

OPÇÃO 5 - RESETAR SENHA ROOT:
No painel Hostinger:
- VPS -> Gerenciar -> Resetar Senha Root
- Aguardar email com nova senha
- Tentar SSH novamente

ARQUIVOS PREPARADOS:
- Pasta 'dist' com build atualizado
- Arquivos modificados: useAuth.tsx, authUtils.ts, VerifyPhone.tsx, etc.

DEPOIS DE ACESSAR O SERVIDOR:
pm2 restart barberpro
pm2 save
